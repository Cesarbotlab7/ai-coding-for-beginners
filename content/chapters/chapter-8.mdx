## 这一章不一样

前面每一章，都是在"认识"——认识前端、后端、数据库、GitHub、服务器……

这一章开始做。

你要做的东西：**一个跑在你电脑上的 AI 资讯页面**。有几张资讯卡片，有一个"AI 总结"按钮，点了之后显示一段摘要。

用到的技术：Astro（就是这个教程网站用的框架）。不需要真数据库，不需要真爬虫，不需要真 AI API——先用假数据让它跑起来，这是所有真实项目最开始的样子。

目标：**让它在你的浏览器里打开，你能看到自己写的东西**。

## 遇到问题怎么办

**报错了？**

把整段报错文字（不只是最后一行）完整复制，粘给 Claude Code，说"我按教程做，跑到这一步报错了，帮我看看"。

**npm 装不上？**

大多数是网络问题。可以试试切换 npm 镜像：
```
npm config set registry https://registry.npmmirror.com
```
然后重新 `npm install`。

**端口被占用？**

改 `astro.config.mjs` 里的端口号：
```js
server: { port: 4322 }
```

**完全跑不动了？**

重启终端，确认自己在正确的文件夹里（`pwd` 命令可以显示当前路径），然后重新试。

## 你做的这个东西，就是真实项目的出生时的样子

AI 资讯平台的第一个版本，也是这样的：一个 HTML 页面、几条假数据、一个不真正调 AI 的"摘要"按钮。

从那里开始，慢慢加上了 Python 爬虫、Supabase 数据库、GitHub Actions 定时任务、DashScope AI 摘要……

每一步都是在上一步跑通之后才加的。

**先让它活着，再让它聪明。**
